Definisce la presentazione del documento HTML
- separa il documento (HTML) dallo stile (CSS)



#### Dove scrivo il CSS?
Tre modi
- EXTERNAL STYLE SHEET
	- creo un file .css e lo linko nell'head dell'HTML

- INTERNAL STYLE SHEET
	- nell'head uso i tag `<style>...</style>`

- INLINE
	- li scrivi direttamente negli elementi 
		- `<h1 style=....>...</h1>`
	- ha fatto un pippone su javascript su metterlo negli elementi (non so se positivamente o negativamente)

### Come è fatta una regola CSS
```css
h1 {
	color: green;
	font-size: 32px;
}
```
- h1 -> selettore
- color: green; -> dichiarazione
- font-size: 32px; -> dichiarazione
- color / fond-size -> proprietà
- green / 32px -> valore

Il selettore dire al browser dove applicare le regole

### Selettori
- selezione tutti gli elementi di quel tipo
	- p
	- body
	- h1
	- ...

Posso anche usare le classi per dare uno stile particolare a un gruppo particolare di elementi
```css
p{
    color: red;
    font-size: 18px;
}

p.verde{
    color: green;
    font-size: 20px;
}
```

nell'html
```html
<p>...</p> -> sarà rosso / 18px

<p class="verde">...</p> -> sarà verde / 20px
```


### Selettore id
Individuare uno e un solo elemento

L'attributo id serve per trovare nella pagina uno e un solo elemento


### Selettori composti
1. Selettore per discendenti (spazio)
	- `A B` -> seleziona i discendenti B di A

2. Selettori per i figli
	- `A > B`

3. Selettori per Fratelli adiacenti (+)
	- `A + B`

4. Selettori per fratelli generici
	- `A ~ B`

## Specificità
La forza con cui un selettore combatte contro altri selettori
Non tutti i selettori hanno lo stesso peso.
Questa specificità è **un insieme di numeri**
- i numeri sono un pezzo della **tupla** `[a,b,c,d]`
	- a=1 -> regola è inline
	- b -> numero di selettori id
	- c -> numero di selettori di classe, attributo o pseudoclasse
	- d -> numero di selettori elemento o pseudo elemento
