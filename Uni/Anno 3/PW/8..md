## Testo, font e colori in CSS
## Font family
Specifica il font utilizzato.
Scriviamo di solito più font, così diamo al sistema una sequenza di font da utilizzare -> parte dal primo e se non lo trova scorre la sequenza
- generalmente l'ultimo deve essere generico (e ha la lettera piccola)

I font si dividono in famiglie

- possiamo importare un file da un server esterno
	- `@font-face`

```css
@font-face {
	font-family: 'Roboto';
	src: url('Roboto-Light-webfont.woff') format ('woff');
}

body {
	font-family: 'Roboto', sans-serif;  /* Add 'Roboto' here */
}
```


I file `.woff` sono contenitori binari che contengono "boh non ho capito" e lo passa al sistema.


## Unità di misure relative
Generalmente non utilizziamo i pixel ma scegliamo
- `%` -> proporzione rispetto al valore ereditato
- `em` -> multiplo del font corrente

>[!tip] Sono relative al font ereditato
>Tipo se ho `Verdana` e voglio 150% vuol dire che lo voglio leggermente più grandi del normale.
