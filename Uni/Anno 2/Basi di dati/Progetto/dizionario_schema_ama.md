
# Dizionario dei Dati - Schema AMA

| ENTITÀ       | DESCRIZIONE                                                           | ATTRIBUTI                                                                                                                                                                                                                                        | CHIAVI PRIMARIE     |
| ------------ | --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------- |
| Prenotazione | Contiene tutte le informazioni relative a una prenotazione effettuata | Codice_prenotazione (PK), Foto_rifiuto, Descrizione_oggetto, Cliente (FK), Lavoratore_preso_in_carico (FK), Tipo_prenotazione, Data_prenotazione, Orario_prenotazione, Veicolo (FK), Sede_associata (FK), Stato_prenotazione, Costo_prenotazione | Codice_prenotazione |
| Valutazione  | Valutazioni associate a prenotazioni concluse                         | Id_valutazione (PK), Codice_prenotazione (FK), Voto (1-5), Stato_prenotazione (unique)                                                                                                                                                           | Id_valutazione      |
| Cliente      | Persona registrata nel sistema come cliente                           | Codice_fiscale (PK), Nome, Cognome, E-mail, Password, Data_di_nascita, Indirizzo_domicilio, CAP_domicilio, Token_SPID, Telefono                                                                                                                  | Codice_fiscale      |
| Lavoratore   | Dipendente dell’AMA con ruolo specifico                               | CID (PK), Nome, Cognome, E-mail, Password, Data_di_nascita, Ruolo                                                                                                                                                                                | CID                 |
| Turno        | Contiene i turni assegnati ai lavoratori                              | Id_turno (PK), Lavoratore (FK), Data_turno, Ora_inizio, Ora_fine, Pausa_inizio, Pausa_fine                                                                                                                                                       | Id_turno            |
| Veicolo      | Veicolo disponibile o in uso per le prenotazioni                      | Targa (PK), Tipologia, Carico_massimo, Stato, Lavoratore_assegnato (FK), Prenotazione (FK)                                                                                                                                                       | Targa               |
| Sede AMA     | Sede operativa AMA con lavoratori assegnati                           | Codice_sede (PK), CAP, Indirizzo, Tabella_orari (FK), Lavoratori (FK)                                                                                                                                                                            | Codice_sede         |
| CAP serviti  | Associa CAP ai territori serviti da ciascuna sede                     | Id_cap (PK), Codice_sede (FK)                                                                                                                                                                                                                    | Id_cap              |
