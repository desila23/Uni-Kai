# STORIA DI JAVA
- Java nasce per astrarre la programmazione e semplificare lo sviluppo di software portabile.  
- Tutti gli oggetti in Java sono gestiti tramite **riferimenti** (non si manipolano direttamente i puntatori come in C/C++).  
- Java è una **implementazione completa della programmazione a oggetti** ad alto livello.  

---

## Java e la sua portabilità
- In C/C++ il codice sorgente va ricompilato su ogni piattaforma perché il compilatore è specifico per l’hardware/OS.  
- Java introduce la **Java Virtual Machine (JVM)** che esegue il **bytecode** indipendentemente dal sistema operativo.  
	- Principio: **Write Once, Run Anywhere** (scrivi una volta, esegui ovunque).  
- Il file `.java` viene compilato da **javac** in un file `.class` (bytecode).  
- La JVM interpreta (e, grazie al JIT compiler, traduce in tempo reale) il bytecode in linguaggio macchina.  
- Il file `.class` può essere eseguito su Windows, Linux, Mac, ecc.  

---

## Caratteristiche di Java
- **Portabilità**: eseguibile su più piattaforme senza ricompilare.  
- **Prestazioni**: non è veloce quanto il C, ma è progettato per un buon compromesso tra efficienza e semplicità.  
- **Sicurezza**: la JVM fornisce isolamento e controlli, anche se non garantisce la sicurezza assoluta.  
- **API ricche**: Java fornisce librerie standard (collezioni, I/O, rete, ecc.).  
- **Uso attuale**: oggi Java è usato soprattutto in applicazioni **back-end** (server, enterprise, Android), mentre gli applet (eseguibili in browser) sono ormai obsoleti.  

---

## Strumenti Java
- **java**: l’interprete che esegue il bytecode.  
- **javac**: il compilatore che converte il sorgente in bytecode.  
- **javadoc**: genera automaticamente documentazione HTML a partire dai commenti nel codice.

# ESEMPIO: Hello World
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

### Note:
- La classe deve avere lo **stesso nome del file** (`HelloWorld.java`).  
- `main` è **public static void** e prende come parametro un array di stringhe (`String[] args`).  
- `System.out.println` stampa a schermo.  

---

## API (Application Programming Interface)
Le API Java sono insiemi di classi, interfacce e pacchetti che forniscono funzionalità già pronte.

Esempi:  
- `java.util`: collezioni di dati (liste, set, mappe).  
- `java.io`: input/output da file e flussi.  
- `java.net`: programmazione di rete (socket, URL, HTTP).  

### Esempio con `ArrayList`
```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> lista = new ArrayList<>();
        lista.add("Elemento 1");
        lista.add("Elemento 2");
        lista.add("Elemento 3");

        for (String elemento : lista) {
            System.out.println(elemento);
        }
    }
}
```
